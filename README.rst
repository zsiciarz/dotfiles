===========
Vim plugins
===========

Vundle
======

::

    mkdir -p ~/.vim/bundle
    git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Afterwards open vim and run ``:BundleInstall``. Done!
